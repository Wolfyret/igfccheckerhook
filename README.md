# igfccheckerhook
A python program that check if indiegala feudalife challenge is currently playable and sends discord hooks.

!!WARNING!! This program is at very alpha stage, and may not function correctly.



## Installation and running

You will need python3 and pip installed on your local system.

Clone the project

```bash
  git clone https://github.com/Wolfyret/igfccheckerhook.git
```

Go to the project directory

```bash
  cd igfccheckerhook
```

Install python libraries

```bash
  pip install -r requirements.txt
```

Run it  :D

```bash
  python3 py.py
```
